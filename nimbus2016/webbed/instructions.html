Rules And Regulations


  • 1. Participants have to create webpages based on the themes given on the home page. He/she can choose only one out of given themes.

  • 2.  Any use of web development softwares like bootstrap, dreamweaver or use of online portals is strictly prohibited.

  • 3. Use of only HTML, CSS, Javascript is encouraged and but of course copying of web pages from any Source would result in immediate termination from contest.

  • 4. Only manually coded web pages would be accepted.

  • 5. Crafting of excessive web pages are not encouraged.

  • 6. Crafted pages are to be submitted to the Team.exe via mail on exewebbed@gmail.com .

  • 7. Only a single entry would be accepted from an individual contestant.

  • 8. Use of web templates is not allowed.

  • 9. The entries should atleast contain a minimum considerable amount of web pages.

  • 10. Jugding criteria would be majorly on creativity and innovativeness rather than quantity.
